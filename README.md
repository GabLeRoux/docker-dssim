# Docker image for [dssim](https://github.com/pornel/dssim)

## Running

```bash
docker run --rm -v ./data:/data awerlang/dssim /data/file-original.png /data/file-modified.png
```

## Usage

Refer to [pornel/dssim](https://github.com/pornel/dssim) usage.
